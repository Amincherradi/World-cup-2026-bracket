## How to work this list
- Send **5 at a time**, not a mass blast. Personalize the first line each time.
- Prefer **LinkedIn DM** to the named role; use email only if you find it.
- Log each in the tracker below. **Follow up once after ~5 days.**
- Lead with the **demo link + a 20s screen recording** of a live match updating.
- Best odds first: **data companies → fantasy/betting → broadcasters → FIFA last.**
